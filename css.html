text-decoration: 종류


none : 글자의 장식을 지정하지 않고, 기본 스타일로 출력된다.
underline : 텍스트에 밑줄을 지정
overline : 텍스트에 윗줄을 지정
line-through : 텍스트 가운데 줄이 그어지도록 지정
blink : 텍스트를 깜빡거리게 지정(현재는 없어진듯)
inherit:부모속성 물려받음

css글자크기조절


font-size:종류

1.기본값 medium
2.medium : 웹브라우저에서 정한 기본 글자크기입니다.
  xx-small, x-small, small, large, x-large, xx-large : medium에 대한 상대적인 크기입니다.
  smaller, larger : 부모 요소의 글자 크기에 대한 상대적인 글자 크기입니다.
  length : px, %, em, rem 등으로 크기를 정합니다.
  initial : 기본값으로 설정합니다.
  inherit : 부모 요소의 속성값을 상속받습니다.
3.px : 픽셀
  %, em : 부모 요소에 대한 상대적인 크기, 각각 100%, 1em이 부모요소의 크기
  rem : html요소의 글자 크기에 대한 상대적인 글자 크기

효과의 발동 우선순위


id 선택자>class 선택자>태그 선택자 / 동일한 선택자라면 나중에 쓰인것이 적용된다.


1. id 선택자
  문서에서 1번밖에 쓸 수 없음, 사용형태 ex) #a{} - id="a"
2. class 선택자
  집단을 지정할 수 있음, 사용형태 ex) .a{} - class="a"
3. 태그 선택자
  일반적인 css효과, 사용형태 a{} - 모든 a태그에 적용
