font-family로 글꼴 종류 지정가능

폰트 예시

font-family: "Times New Roman", Times, serif;
처음 폰트가 컴퓨터에 있으면 처음 폰트 지정, 없으면 순서대로 지정, 마지막 폰트는 포괄적인 폰트로 지정

포괄적인 폰트들

font-family : serif (장식이 있는 폰트)
font-family : sans-serif
font-family : cursive (흘림체)
font-family : fantasy
font-family : monospace (고정폭)

한글폰트

폰트굵게
font-weight: 로 폰트 굵기 설정가능, 주로 bold(굵게)를 이용함
line-height: 로 줄간격 설정가능, 1.2가 기본

폰트의 여러 속성 한번에 지정하기

순서를 지켜야하고 font-size와 font-family는 필수
font: font-style font-variant font-weight font-size/line-height font-family|caption|icon|menu|message-box|small-caption|status-bar|initial|inherit;
