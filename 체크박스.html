input태그의 type들

  • text : 글로 입력
  • number : 숫자로 입력
  • radio : 원형 체크박스,같은 name들 중 단일선택만 가능
  • checkbox : 체크박스, 다중선택 가능
  • submit : 제출기능
  • reset : 리셋기능
  • button : 버튼 생성
  • image : 이미지버튼 생성
  • file : 파일 선택필드 생성
  • hidden : 감춰진 필드
